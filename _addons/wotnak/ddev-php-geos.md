# ddev-php-geos

[DDEV addon](https://ddev.readthedocs.io/en/latest/users/extend/additional-services/) that installs [php-geos](https://git.osgeo.org/gitea/geos/php-geos) module.

## Installation

```bash
ddev add-on get wotnak/ddev-php-geos
ddev restart
```
