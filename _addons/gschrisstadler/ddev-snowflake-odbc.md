# DDEV Snowflake ODBC

## Install/Update

For DDEV v1.23.5 or above run

```bash
ddev add-on get gschrisstadler/ddev-snowflake-odbc
```

For earlier versions of DDEV run

```bash
ddev get gschrisstadler/ddev-snowflake-odbc
```

Then restart your project

```bash
ddev restart
```
